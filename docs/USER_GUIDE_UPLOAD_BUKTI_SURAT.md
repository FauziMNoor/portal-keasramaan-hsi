## ğŸ¯ Tips & Best Practices

### Untuk Kepala Asrama:
1. **Screenshot yang Baik:**
   - Pastikan formulir terlihat jelas
   - Tidak blur atau gelap
   - Semua bagian penting terlihat
   - Tanda tangan wali terlihat

2. **Format File:**
   - JPG: Untuk foto dari HP (lebih kecil)
   - PNG: Untuk screenshot dari komputer
   - PDF: Untuk scan dokumen

3. **Ukuran File:**
   - Jika >5MB, compress dulu
   - Gunakan tools online: tinypng.com, compressjpeg.com
   - Atau resize gambar ke 1920x1080

### Untuk Kepala Sekolah:
1. **Verifikasi Bukti:**
   - Zoom gambar untuk detail
   - Cek tanda tangan wali
   - Cek tanggal formulir
   - Cek kesesuaian data

2. **Surat Izin:**
   - Download segera setelah approve
   - Print dengan kualitas baik
   - Gunakan kertas A4
   - TTD dengan tinta hitam/biru

### Untuk Admin:
1. **Update Data:**
   - Update setiap awal tahun ajaran
   - Update saat ada pergantian pejabat
   - Cek data sebelum cetak surat pertama kali

## â“ FAQ (Frequently Asked Questions)

### Q: Apakah wajib upload bukti formulir?
**A:** Ya, wajib untuk Kepala Asrama saat approve. Sistem tidak akan mengizinkan approve tanpa upload bukti.

### Q: Format file apa yang diterima?
**A:** JPG, PNG, dan PDF. Maksimal 5MB.

### Q: Bagaimana jika file terlalu besar?
**A:** Compress file menggunakan tools online atau resize gambar.

### Q: Apakah bisa upload ulang jika salah?
**A:** Ya, bisa upload ulang sebelum Kepala Sekolah approve.

### Q: Bagaimana jika bukti tidak jelas?
**A:** Kepala Sekolah bisa tolak perizinan dan minta Kepala Asrama upload ulang dengan bukti yang lebih jelas.

### Q: Apakah surat izin bisa diedit setelah didownload?
**A:** Tidak, surat izin di-generate otomatis dan tidak bisa diedit. Jika ada kesalahan data, harus edit perizinan dan generate ulang.

### Q: Bagaimana jika data info sekolah salah di surat?
**A:** Update data di halaman "Info Sekolah", lalu generate ulang surat izin.

### Q: Apakah surat izin tersimpan di sistem?
**A:** Tidak, surat di-generate on-demand setiap kali download. Jadi bisa download berkali-kali.

### Q: Bagaimana jika lupa download surat?
**A:** Bisa download kapan saja selama status masih "Disetujui". Tombol download akan selalu tersedia.

### Q: Apakah bisa download surat sebelum disetujui Kepala Sekolah?
**A:** Tidak, surat hanya bisa didownload setelah status "Disetujui" (approved oleh Kepala Sekolah).

## ğŸ†˜ Troubleshooting

### Upload Gagal
**Masalah:** File tidak bisa diupload
**Solusi:**
1. Cek ukuran file (<5MB)
2. Cek format file (JPG/PNG/PDF)
3. Cek koneksi internet
4. Refresh halaman dan coba lagi

### Preview Tidak Muncul
**Masalah:** Gambar bukti tidak muncul
**Solusi:**
1. Refresh halaman
2. Cek koneksi internet
3. Coba browser lain
4. Clear cache browser

### PDF Tidak Ter-generate
**Masalah:** Surat izin tidak bisa didownload
**Solusi:**
1. Pastikan status "Disetujui"
2. Pastikan data info sekolah sudah diisi
3. Refresh halaman
4. Coba lagi

### Data Surat Salah
**Masalah:** Data di surat tidak sesuai
**Solusi:**
1. Update data di "Info Sekolah"
2. Generate ulang surat
3. Atau edit data perizinan jika data santri yang salah
