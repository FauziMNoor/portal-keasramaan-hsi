# Cara Hitung Persentase - Penjelasan Sederhana

## Rumus Dasar

```
Persentase = (Total Poin yang Didapat / Total Poin Maksimal) × 100%
```

## Contoh Perhitungan

### 1. UBUDIYAH = 75%

**Indikator Ubudiyah (8 indikator):**
```
1. Shalat Fardhu Berjamaah    : max 3 poin
2. Tata Cara Shalat            : max 3 poin
3. Qiyamul Lail                : max 3 poin
4. Shalat Sunnah               : max 3 poin
5. Puasa Sunnah                : max 5 poin
6. Tata Cara Wudhu             : max 3 poin
7. Sedekah                     : max 4 poin
8. Dzikir Pagi Petang          : max 4 poin
─────────────────────────────────────────
TOTAL MAKSIMAL                 : 28 poin
```

**Poin yang Didapat (rata-rata periode):**
```
1. Shalat Fardhu Berjamaah    : 2.8 poin
2. Tata Cara Shalat            : 2.5 poin
3. Qiyamul Lail                : 2.0 poin
4. Shalat Sunnah               : 2.3 poin
5. Puasa Sunnah                : 3.5 poin
6. Tata Cara Wudhu             : 2.7 poin
7. Sedekah                     : 3.0 poin
8. Dzikir Pagi Petang          : 2.2 poin
─────────────────────────────────────────
TOTAL DIDAPAT                  : 21.0 poin
```

**Perhitungan:**
```
Persentase = (21.0 / 28) × 100%
           = 0.75 × 100%
           = 75%
```

## Cara Mendapatkan "Poin yang Didapat"

Poin yang didapat adalah **RATA-RATA** dari semua data dalam periode:

### Contoh: Shalat Fardhu Berjamaah (max 3 poin)

**Data 5 hari:**
```
Hari 1: 3 poin
Hari 2: 2 poin
Hari 3: 3 poin
Hari 4: 3 poin
Hari 5: 2 poin
```

**Rata-rata:**
```
Rata-rata = (3 + 2 + 3 + 3 + 2) / 5
          = 13 / 5
          = 2.6 poin
```

Jadi untuk indikator "Shalat Fardhu Berjamaah", poin yang didapat adalah **2.6 poin** dari maksimal **3 poin**.

---

## Catatan Penting

1. **Periode**: Perhitungan bisa untuk 30 hari atau 1 semester
2. **Rata-rata**: Semua nilai dirata-ratakan dulu, baru dijumlahkan
3. **Nilai 0**: Jika ada hari tidak ada data (nilai 0), tidak dihitung dalam rata-rata
4. **Pembulatan**: Hasil akhir dibulatkan ke bilangan bulat

---

## Interpretasi

- **75-100%**: Sangat Baik (Mumtaz)
- **60-74%**: Baik (Jayyid Jiddan)
- **50-59%**: Cukup (Jayyid)
- **30-49%**: Kurang (Dhaif)
- **0-29%**: Sangat Kurang (Maqbul)
